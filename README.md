## 🚀 Quick Start

### **1. Clone Repository**
```bash
git clone https://github.com/MarvXLab/ghydra-ai.git
cd ghydra-ai
```

### **2. Install Dependencies**
```bash
pip install -r requirements.txt
```

### **3. Train AI Model**
```bash
python train_sklearn.py
```

### **4. Launch Platform**
```bash
streamlit run streamlit_app.py
```

### **5. Access Dashboard**
Open http://localhost:8501 in your browser
